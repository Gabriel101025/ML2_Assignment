# ML2 Assignment

## Overview
This project tracks ML experiments with MLflow, analyzes data drift, and automates retraining using GitHub Actions.

## Repository Structure
- `src/` : Core scripts for training and drift evaluation
- `tests/` : Unit tests (`test_model.py`)
- `data/` : Datasets (`day_2011.csv`, `day_2012.csv`)
- `requirements.txt` : Dependencies
- `best_model.pkl` : Saved best model
- `README.md` : Instructions
